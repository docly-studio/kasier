# Kasier — Modern POS Web App

Kasier adalah aplikasi kasir (Point of Sale) modern berbasis web yang dibuat khusus untuk warung, toko kecil, UMKM, dan bisnis rumahan. Aplikasi ini menghadirkan pengalaman kasir yang ringan, cepat, responsif, dan mudah digunakan langsung dari browser.

Dengan tampilan mobile-first modern, Kasier dirancang agar nyaman digunakan di smartphone maupun desktop tanpa memerlukan aplikasi tambahan.

---

# ✨ Overview

Kasier menggabungkan sistem kasir, manajemen produk, laporan penjualan, dan pengaturan toko dalam satu dashboard sederhana namun powerful.

Aplikasi ini menggunakan:

* Frontend modern berbasis HTML + TailwindCSS
* Backend Google Apps Script
* Google Sheets sebagai database

Tidak memerlukan server database terpisah sehingga lebih ringan dan praktis untuk UMKM.

# 🛒 Fitur Utama

## Kasir Modern

* Tampilan kasir cepat dan responsif
* Tambah produk ke keranjang
* Hitung total otomatis
* Hitung kembalian otomatis
* Support pembayaran:

  * Tunai
  * QRIS
  * Transfer Bank

## Scan Barcode / SKU

* Scan barcode langsung dari kamera
* Pencarian produk berdasarkan SKU
* Auto generate SKU produk

## Manajemen Produk

* Tambah produk
* Edit produk
* Hapus produk
* Manajemen stok
* Kategori produk
* Upload gambar produk

## Dashboard Statistik

* Statistik pendapatan harian
* Grafik penjualan 7 hari terakhir
* Grafik kategori produk terlaris
* Perhitungan keuntungan otomatis

## Riwayat Transaksi

* Menyimpan seluruh histori penjualan
* Detail item transaksi
* Cetak ulang struk

## Pengaturan Toko

* Nama toko
* Alamat toko
* Nomor WhatsApp
* Custom warna tema
* Upload QRIS toko

# 📱 User Interface

Kasier menggunakan konsep:

* Mobile First UI

* Bottom Navigation Style
* Smooth Animation
